# Movie Recommendation System

A content-based movie recommendation system that suggests similar movies based on genres and plot descriptions. The project uses Natural Language Processing (NLP) techniques and provides an interactive web interface built with Streamlit.

---

##  Project Overview

This application recommends movies by analyzing textual features such as genres and movie overviews. It uses TF-IDF vectorization and cosine similarity to find movies with similar content. Additionally, the app integrates the TMDB API to display movie posters and ratings when available.

---

##  Features

- Content-based movie recommendations  
- TF-IDF vectorization for text processing  
- Cosine similarity for measuring movie similarity  
- Interactive Streamlit web interface  
- Adjustable number of recommendations  
- Movie overview display  
- Movie poster and rating integration using TMDB API  
- Graceful handling of API unavailability  

---

##  Technologies Used

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit  
- TMDB API  
- Requests
